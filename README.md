# 👋 SignSpeak - Penerjemah Bahasa Isyarat Berbasis AI

SignSpeak adalah aplikasi web inovatif yang dirancang untuk menerjemahkan bahasa isyarat tangan secara real-time menjadi teks dan suara. Project ini dibangun untuk menjembatani komunikasi antara teman tuli dan masyarakat umum.

## 🚀 Fitur Utama
- **Real-Time Detection**: Menggunakan kamera HP/Laptop untuk mendeteksi gestur tangan langsung.
- **Cloud History**: Menyimpan hasil terjemahan penting ke database Google Cloud Firestore.
- **Text-to-Speech (TTS)**: Mengonversi hasil teks terjemahan menjadi suara agar bisa didengar.
- **Responsive Design**: Tampilan antarmuka modern yang nyaman dibuka di perangkat apa pun.

## 🛠️ Teknologi yang Digunakan
- **Frontend**: HTML5, Tailwind CSS v4 (Browser Compiler)
- **Artificial Intelligence**: MediaPipe Hands (Google Edge AI)
- **Database**: Google Firebase Firestore (Compat Version)
- **Hosting**: Firebase Hosting
